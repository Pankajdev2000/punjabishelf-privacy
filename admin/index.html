📖 Field Guide
What each field means and tips for filling them correctly
📝 Fields Reference
FieldRequiredTips
Title (English)✅ YesUse the commonly known name. e.g. "Raseedi Ticket"
Title Gurmukhi✅ YesPaste from Google Translate or type in Punjabi keyboard. e.g. ਰਸੀਦੀ ਟਿਕਟ
Author✅ YesFull name in English. e.g. "Amrita Pritam"
Description✅ Yes2–4 sentences about the book. Shown on book detail screen.
PagesNoCount from the PDF. Leave 0 if unknown.
Language✅ YesThe language the book is written in — not the reader's language.
Category✅ YesPick the closest match. Poetry = kavita, Biography = jeevni etc.
Type: Free—All users can read without paying. Good for classics & samples.
Type: Premium—Only ₹29/month subscribers can read. Use for new or popular books.
Hard CopyNoOnly enable if you can actually source and ship this book. Premium users only.
Cover Image✅ YesJPG/PNG. Try to get the actual book cover. Min 300×400px recommended.
PDF File✅ YesMake sure the PDF has proper Gurmukhi fonts embedded, not scanned images (for best reading experience).
💡 Tips
  • 🌐 Gurmukhi typing: Use Google Input Tools (translate.google.com) to type Punjabi text, then copy-paste it here.
  • 📄 PDF quality: Text-based PDFs read much better than scanned image PDFs in the Syncfusion viewer.
  • 🖼️ Cover images: Search "[book name] book cover" on Google Images. Right-click → Save.
  • 🚀 After upload: Books appear in the app instantly — no rebuild needed. Pull to refresh in the app.
  • 🗑️ Delete: Deleting from this panel removes the Firestore document. The PDF/cover in Storage is NOT deleted (to avoid broken links in reading history). Delete manually from Firebase Storage if needed.
